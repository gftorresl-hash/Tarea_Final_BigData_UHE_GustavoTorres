# 🤖 AI Student Impact Analysis Dashboard

## 📊 Análisis del Impacto de la Inteligencia Artificial en el Rendimiento Académico de Estudiantes

Aplicación interactiva desarrollada con **Streamlit** para analizar cómo el uso de herramientas de Inteligencia Artificial influye en el rendimiento académico de estudiantes.

El proyecto integra técnicas de:

* Análisis exploratorio de datos (EDA).
* Visualización interactiva.
* Estadística descriptiva.
* Machine Learning.
* Predicción del rendimiento académico.
* Generación automática de reportes.

---

# 🎯 Objetivo del Proyecto

El objetivo principal es construir una plataforma analítica que permita:

* Explorar patrones relacionados con el uso de Inteligencia Artificial.
* Identificar factores asociados al desempeño académico.
* Visualizar relaciones entre variables educativas.
* Crear modelos predictivos basados en Machine Learning.
* Facilitar la toma de decisiones mediante análisis de datos.

---

# 🚀 Características Principales

## 📊 1. Análisis Exploratorio

Incluye:

* Información general del dataset.
* Tipos de variables.
* Valores faltantes.
* Estadísticas descriptivas.
* Distribución de variables.

---

## 📈 2. Visualizaciones Interactivas

La aplicación permite analizar:

* Distribuciones de datos.
* Comparaciones entre grupos.
* Relaciones entre variables.
* Tendencias del comportamiento estudiantil.

Las visualizaciones fueron desarrolladas utilizando:

* Plotly.
* Matplotlib.

---

## 🤖 3. Modelo Predictivo

Se implementó un modelo de Machine Learning utilizando:

**Random Forest Regression**

Funciones:

* División automática entrenamiento/prueba.
* Entrenamiento del modelo.
* Predicción de resultados académicos.
* Evaluación mediante métricas.

Métricas utilizadas:

* MAE (Mean Absolute Error).
* RMSE (Root Mean Squared Error).
* R² Score.

Además incluye:

* Ranking de importancia de variables.
* Comparación entre valores reales y predichos.

---

## 📥 4. Reportes y Exportación

La aplicación permite:

* Descargar dataset procesado.
* Exportar estadísticas descriptivas.
* Generar reportes automáticos.
* Preparar resultados para análisis posteriores.

# 🌐 Despliegue en Streamlit Cloud

Para publicar la aplicación:

1. Subir el proyecto a GitHub.
2. Crear una cuenta en Streamlit Cloud.
3. Seleccionar el repositorio.
4. Elegir:

```
app.py
```

como archivo principal.
5. Ejecutar Deploy.
